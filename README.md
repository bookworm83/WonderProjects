The goal of this project is to analyse a movie database and develop a data-driven approach to help users choose what to watch among highly-rated movies when decision fatigue is high.

I designed rule-based movie recommendations using SQL logic and Python, focusing on explainability rather than machine learning.

Dataset

IMDb Top 1000 Movies 

Source: Kaggle https://www.kaggle.com/datasets/omarhanyy/imdb-top-1000?resource=download 

All information is on one big table containing the following columns:

Title, Certificate, Duration, Genre, Rate, Metascore, Description, Cast, Info

Note: the minimum rate in raw data is 8.0
Note: oldest movie in raw data was released in 1920
Note: newest movie in raw data was released in 2020
